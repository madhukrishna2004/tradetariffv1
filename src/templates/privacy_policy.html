Limited Use Disclosure

TradeSphere Global’s use and transfer of information received from Google APIs will adhere to the Google API Services User Data Policy , including the Limited Use requirements.

1. Introduction

This Privacy Policy describes how KrisLynx LLP, operating under TradeSphere Global, collects, uses, and protects your information when interacting with our services, including integrations using Google OAuth and Gmail APIs.

2. Information We Collect

  • Basic identifiers: Name, email address (only after explicit OAuth consent).
  • OAuth authorization: We request the Gmail send permission to enable automated email sending actions initiated by you.

3. Use of Gmail API

We only use the Gmail API with your consent to send transactional emails on your behalf. For example, confirmations, alerts, or notifications triggered by your actions on the platform. We do not read, access, or store your inbox, emails, or personal Gmail data.

4. Google API Services Compliance

Our application’s use of Google APIs, including Gmail and other Google Workspace APIs, strictly adheres to the Google API Services User Data Policy , including its Limited Use requirements. We only use access to Google user data to provide or improve user-facing features that are prominent in the requesting application's user interface.

We affirm that our use of Google user data is compliant with all applicable Google policies and requirements, including:

  • Limited Use of User Data requirement under the Google Workspace API User Data and Developer Policy

We do not transfer, sell, or share user data with third parties. All access and usage of Google user data is limited strictly to user-initiated actions, such as sending emails via the Gmail API, in accordance with the intended functionality of our application.

5. Data Security

All data transmission is protected using HTTPS and OAuth 2.0 industry-standard encryption. Gmail tokens are securely handled and never stored beyond the session unless required for enterprise login continuity with user consent.

6. User Rights

  • You can revoke access at any time via your Google Account Permissions.
  • You may request deletion of any stored data related to your session by emailing us.
