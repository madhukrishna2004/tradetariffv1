Privacy Policy

Your privacy is our priority. Read how we handle your data securely.

1. Introduction

Welcome to **TradeSphere Global**. This Privacy Policy explains how we collect, use, and protect your information when using our services.

✦ ✦ ✦

2. Information We Collect

We only collect **essential user data** required for service functionality. This includes:

  • OAuth authentication details (temporary access tokens, never stored).
  • User-provided data when using Google Forms, Gmail, or Sheets integrations.
  • Basic account details (name, email) for authentication purposes.
✦ ✦ ✦

3. How We Use Your Data

We use the collected data **only for its intended purpose** and never share or sell it. This includes:

  • Sending **user-initiated emails** via Gmail API.
  • Creating and modifying Google Forms **on user request**.
  • Storing structured data in Google Sheets **with user authorization**.
✦ ✦ ✦

4. Google API Compliance

TradeSphere Global complies with **Google’s API Services User Data Policy** and **Limited Use Policy**:

  • OAuth tokens are never stored beyond the active session.
  • Data is never shared with third parties.
  • We request the minimum necessary permissions for app functionality.
✦ ✦ ✦

5. Data Security

We implement security measures to protect your information:

  • All communication is **encrypted** using secure protocols.
  • We do not store or log OAuth tokens after authentication.
  • User data is only processed **when explicitly requested** by the user.
✦ ✦ ✦

6. User Rights & Choices

You have the right to:

  • Revoke OAuth permissions at any time via **Google Account settings**.
  • Request deletion of any data associated with your account.
✦ ✦ ✦

7. Changes to Privacy Policy

We may update this Privacy Policy as needed. Changes will be notified through our platform.

✦ ✦ ✦

For questions or data requests, contact us at support@tradesphere.com
